## Нужны доработки в части

В элементах шаблона используется путь API:
`https://{HOST.CONN}/redfish/v1/Systems/1/Storage/RAID_Slot10/Drives/Disk.15`

Соответственно если название в шасси у рейда (RAID_Slot10) будет другое, то api не отдаст нужные данные

Надо изменить шаблон и использовать RAID_Slot10 как заранее определённую переменную
